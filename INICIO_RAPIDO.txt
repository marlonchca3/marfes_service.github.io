🚀 DESPLEGAR EN 3 PASOS:

  PASO 1: Instalar Firebase CLI (una sola vez)
  ────────────────────────────────────────────
  npm install -g firebase-tools

  
  PASO 2: Obtener API key de Google Gemini (gratis)
  ──────────────────────────────────────────────────
  1. Abre: https://aistudio.google.com/app/apikey
  2. Haz clic en "Create API key"
  3. Copia tu clave
  4. Pega en: functions/.env
  
  Pasos detallados en: CHECKLIST.md
  

  PASO 3: Desplegar
  ────────────────
  cd functions
  npm install
  cd ..
  firebase deploy --only functions
